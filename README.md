# Projeto Commerce

### Features
- [] Usuário
  - [] Registro
  - [] Autenticação
- [] Produtos
  - [] CRUD de produtos
  - [] Adicionar e Remover fotos
- [] Carrinho
  - [] Adicionar ou Remover items
  - [] Alterar quantidade de um item
- [] Encomenda
  - [] Criação de encomenda (Cartao de credito/ Boleto/ Pix)
  - [] Cancelamento de encomenda
  - [] Webhook para alterar status
- [] Envio de e-mail
